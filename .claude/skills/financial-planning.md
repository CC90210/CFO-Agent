# Atlas Financial Planning & Wealth Skill

## When to activate
Any question about: budget, spending, savings, FIRE, retire, net worth growth, wealth, insurance, estate, succession, incorporation, should I incorporate, how much to save, allocation, emergency fund.

## Quick Commands
```bash
python main.py runway      # Cash flow + runway projections
python main.py rebalance   # Portfolio allocation analysis
```

## Python Engines
- `finance/advisor.py` — Claude-powered financial advisory (risk profiling, rebalancing, emergency planning)
- `finance/budget.py` — expense tracking, monthly summaries, savings suggestions
- `finance/wealth_tracker.py` — net worth over time, wealth projections
- `finance/tax.py` — tax impact modeling for financial decisions

## Rules
- All recommendations must respect the Montreal $10K floor
- CCPC incorporation trigger: $80K+ annual income
- CC's 2026 projection: $280K-$480K CAD — plan for mid-range
- RRSP contributions wrong below $55K marginal rate
- FHSA: opened 2026-03-27, $8K/year room
- Never recommend products Atlas hasn't researched
